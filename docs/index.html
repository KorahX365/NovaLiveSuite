Componentes

Cuatro Overlays,
Un Solo Panel

Cada overlay es una fuente de navegador independiente en OBS. Configura todo en el panel web y los cambios se aplican en vivo sin recargar.

Chat Glassmórfico

Mensajes de Twitch con diseño glassmorfismo. Badges de broadcaster, moderador y suscriptor. Animaciones de entrada/salida configurables.

chat.html · IRC WebSocket

Now Playing + Vinilo

Detecta Spotify, Chrome, Firefox y reproductores locales vía Windows Media Controls. Vinilo animado, visualizador de audio y letras sincronizadas.

nowplaying.html · SMTC API

Alertas con Partículas

Follows, subs y raids con confeti y chispas procedimentales. Cuatro layouts: imagen arriba, laterales o texto superpuesto con fondo tintado.

alerts.html · EventSub WS

Sponsor Carousel

Rota imágenes, vídeos e iFrames con transiciones en 3D: fade, slide y zoom. Perfecto para logos de patrocinadores y contenido promocional.

carousel.html · SSE Stream

System Tray & Autostart

Icono nativo en la bandeja del sistema de Windows. Configura el autoarranque al inicio de sesión mediante el registro de Windows.

Tray Icon · Registry Autostart

Dashboard de Config

Panel web completo con previsualización en vivo, importación/exportación de settings y recarga automática de todos los overlays conectados.

Rust Axum Backend

Servidor HTTP ligero y de alto rendimiento. Los assets HTML/CSS/JS se compilan directamente en el binario con include_str!, resultando en un único ejecutable sin dependencias.

cargo build --release

SSE + WebSockets

Los overlays se suscriben a /api/stream vía Server-Sent Events para recibir actualizaciones de configuración en tiempo real. Los feeds de alertas usan WebSockets para máxima reactividad.

EventSource /api/stream

Twitch EventSub & IRC

Conexión directa vía WebSocket a la API EventSub de Twitch para follows, subs y raids. Chat a través de IRC con parsing de tags y badges.

wss://eventsub.wss.twitch.tv

Windows SMTC

Lee metadatos de medios del sistema operativo (título, artista, álbum, portada) usando la API System Media Transport Controls de Windows. Compatible con Spotify, Chrome, Firefox y más.

GlobalSystemMediaTransportControls
